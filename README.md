# 🎮 John95AC Website Documentation

**My Centralized Documentation Hub for Skyrim SE Mods, Tools, and Development Projects.**

I built this website to serve as a documentation repository for my SKSE DLL mods, Python utilities, and interactive tools designed for the Skyrim Special Edition modding ecosystem.

---

## Overview

This is my personal project where I consolidated all documentation, installation guides, and interactive demonstrations for my Skyrim modding tools into a single, navigable web interface. I built it using HTML5 UP's Landed template to provide a modern dark-theme experience with responsive design and embedded interactive features.

I designed this website for developers and modders who need:
- **Centralized documentation** for multiple SKSE projects
- **Interactive tools** for testing and demonstration
- **Progress tracking** across my development milestones
- **Direct links** to my social platforms and support channels

---

## System Overview

When I built this website, I prioritized:
- **Single-page navigation** with smooth scrolling between sections
- **Embedded tools** for immediate interaction without external dependencies
- **Mermaid.js integration** for visual flowchart documentation
- **Audio visualization** with waveform rendering for my sound-related projects
- **Responsive design** optimized for desktop and mobile viewing

## My Projects

### 🎨 OBody NG Preset Distribution Assistant NG

> [Open Project →](OBody_NG_Preset_Distribution_Assistant_NG/index.html)

This is an SKSE DLL plugin I developed to automate OBody preset distribution using INI configuration files paired with SPID. I built it to eliminate manual JSON editing by processing distribution rules automatically. I included detailed installation guides, Mermaid.js flowcharts for workflow visualization, and comprehensive configuration examples.

**Tech I used:** C++ SKSE Plugin, INI Parsing, JSON Automation

---

### 🔧 RaceMenu Overlay to SlaveTats Helper

> [Open Project →](Race_Menu_Overlay_to_SlaveTats_Helper/index.html)

I created this Python application with PyQt5 for bidirectional conversion between RaceMenu overlay configurations and SlaveTats compatible formats. I designed it for modders who want to convert existing overlays or create new ones compatible with both systems. I included step-by-step guides and architecture documentation.

**Tech I used:** Python, PyQt5, JSON/XML Processing

---

### 🎵 OSoundtracks Expansion Sounds SA

> [Open Project →](OSoundtracks_Expansion_Sounds_SA/index.html)

This is my audio expansion ecosystem for Skyrim SE featuring dynamic soundtrack integration, sound player capabilities, and configurable generator packs. I built it to provide immersive audio layers for gameplay scenes with real-time waveform visualization and playlist management.

**Tech I used:** SKSE Plugin, BASS Audio Library, Web Audio API

---

### 📰 My Projects News

## Quick Start

### Navigate the Website

Open `index.html` in any modern browser. My landing page provides:
- Project showcase with direct links to my documentation
- Navigation menu for quick access to all sections
- My social media links in footer for support and updates

### Use My Interactive Tools

Navigate to `Tools/` folder and open any HTML file:
- **Dado** — Click dice to generate random values
- **Pachinko** — Click to drop balls and observe physics simulation
- **Grim Generator** — Drag-and-drop video files for sprite sheet extraction

### Browse My Project Documentation

Each project folder I created contains:
- `index.html` — Main documentation page
- `style.css` — Project-specific styling
- Supporting assets and scripts
